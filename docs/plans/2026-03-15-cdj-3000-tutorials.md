### Pass 2: Relationship Map

Key dependencies between features:
- Beat Sync requires PRO DJ LINK network (multiple players connected via LAN)
- Beat Sync requires tracks analyzed by rekordbox (beatgrid present)
- Key Sync requires rekordbox analysis AND Beat Sync master established
- Quantize requires rekordbox analysis (beatgrid)
- Beatgrid adjustment (Grid Adjust) interacts with Zoom mode (both via rotary hold)
- Hot Cue AUTO LOAD is set in UTILITY or SHORTCUT and affects playback workflow
- Slip mode interacts with: Scratch (Vinyl mode), Loop, Hot Cue, Reverse, Pause
- DIRECTION lever has 3 states: FWD, REV, SLIP REV — SLIP REV works independently of Slip button
- CUE/LOOP CALL ◄► navigates both saved cues AND saved loops (shared control)
- 4 BEAT LOOP (1/2X) doubles as loop halver during loop playback
- 8 BEAT LOOP (2X) doubles as loop doubler during loop playback
- LOOP IN button doubles as CUE IN ADJUST during loop fine-adjust mode
- LOOP OUT button doubles as OUT ADJUST during loop fine-adjust mode
- Touch display BEAT LOOP panel (on Waveform screen) overlaps with physical 4/8 BEAT LOOP buttons
- MY SETTINGS save/load flows through SHORTCUT screen, not UTILITY directly
- WAVEFORM COLOR (SHORTCUT) and WAVEFORM/PHASE METER toggle are per-device settings
- Tag List can be converted to a rekordbox Playlist — only works if rekordbox library on device

## NOTES FOR TUTORIAL BUILDER
- The CDJ-3000 has NO keyboard, NO synthesis, NO zones. All tutorials are about DJ workflow.
- The 9" touch display is the primary UI surface. Many features have both a physical button AND a touch screen equivalent (e.g. BEAT LOOP buttons vs touching [BEAT LOOP] on waveform screen).
- Panel state for tutorials: the jog display shows artwork + status indicators. The touch display shows one of ~12 distinct screens. Every tutorial step must specify which screen is showing.
- rekordbox analysis is a prerequisite for: Beat Sync, Key Sync, Quantize, Beatgrid editing, Track Filter BPM/Key. Note this in each relevant tutorial.
- PRO DJ LINK (multiple players networked) is a prerequisite for: Beat Sync, Key Sync, Instant Doubles, Tag List sharing, DUPLICATION, Touch Preview, Touch Cue.
- Saved cues/loops: up to 10 per track per storage device (p.85). Relevant to Cue and Loop tutorials.
- Hot Cues: up to 8 per track (A-H) (p.58).
